# ZNGIT WebService

This was made by using LoopBack framework as back-end and AngularJS as front-end.


## Skills
Node.js, AngularJS, LoopBack, MySQL, Stripe

## Site link
http://www.zngit.com

```
$ git clone git@bitbucket.org:smartman87219/zngit-webservice.git
$ cd zngit-webservice
$ npm install
$ node .
```
